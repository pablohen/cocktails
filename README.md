# Cocktails

Exemplo do consumo de uma API e uso do Context API https://cocktails.pablohen.dev.

## Front-end:

- TypeScript
- React
- Context API
- Axios
- Material UI
- React Query
- Vite+

## Back-end:

- [TheCocktailDB](https://www.thecocktaildb.com)

### Comentários:

- Instalação das dependências via `pnpm install` ou `vp install`
- Iniciar com `vp dev` (ou `pnpm run dev`)
