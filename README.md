# Cocktails

Exemplo do consumo de uma API e uso do Context API https://cocktails.pablohen.dev.

## Front-end:

- TypeScript
- React
- Context API
- Axios
- React Icons
- React Query
- Tailwind CSS

## Back-end:

- [TheCocktailDB](https://www.thecocktaildb.com)

### Comentários:

- Instalação das dependências via 'pnpm i'
- Iniciar pelo script 'dev', conforme o que está no 'package.json'
